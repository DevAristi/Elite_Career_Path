# RAM
## Bits & Bytes
B = Byte
b = bit
They can be thought of as a position that can store a digit - the restriction on the ddigit is that it can either be a 0 or a 1.
So individual bits can form multiple bits which can form bytes which can form RAM and RAM can be used to store Advanced data structures.
## Arrays
**For integers:** They are **always going to be contiguous**. (When you are going to store an array in the ram, lets say: 1,3,5 they are going to be store in that order, nothing is going to be between the three)
Each value of an integer takes four bytes to store. Thats why the adreess will be ``0-4-8`` for each integer in the example. (1,3,5)
**For characters like ABC:** Are also **contiguously** each character only takes 1 byte. (8 bits)
The adress for ABC will be ``0-1-2``.
